Группа "ВЯТКА" приложуха

Концепция:

1. Группа имеет цели и планы, которые они вместе проходят

- Цели включают в себя: полевые выходы, грядущие курсы, соревнования и тд
- Планы подразумевают структурированный список действий, который позволит приблизится к выполнению цели
- Группые покупки. К примеру, общая закупка снаряги или сбор на одну крупную вещь

2. Каждый участник группы может указывать свой список снаряжения для:

- Учета
- Возможности определить, что следует обновить
- Указать комплектацию портящихся продуктов (еды, аптеки и тд) с целью своевременного обновления
- Наглядность для командиров

3. Групповая деятельность

- Список рекоммендуемого снаряжения для покупки новичкам и не только
- Небольшие выписки по темам, которые позволят вспомнить основные моменты тренировки

Обобщенно: социальная сеть для партизан/тактикульщиков

Функции

1) Учет снаряжения

- Название
- Фирму (название и ссылку на сайт)
- Фотокарточки
- Группа снаряжения
- Описание/Заметки
- Цена
- Качество (по 10 баллам)

2) План покупки

- Цель-Снаряжение
- Дата, когда нужно купить

HTTP Запросы

1) Warrior

-> POST  /warriors
-> GET   /warriors/{id}
-> POST  /warriors/{id}/inventory
-> POST  /warriors/{id}/tasks

2) Squad

-> POST  /squads
-> GET   /squads/{id}
-> POST  /squads/{id}/warriors
